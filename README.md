# csv2json
csv2json - Python script to generate JSON files from CSV.


```
➜  csv2json git:(master) ✗ ./csv2json.py
***************************
*** Convert CSV to JSON ***
***************************

Directory path or RETURN for current directory:


Selected directory: 
*******************
/csv2json

Choose csv file: 
****************
[1]: /csv2json/name_gender_dataset.csv
[2]: /csv2json/simpsons.csv

Choose file [Enter number]:2

Choose delimiter (Default ;):;

Generate JSON file: 
*******************
Filename Input: /csv2json/simpsons.csv
Filename Output: /csv2json/simpsons.json

```
